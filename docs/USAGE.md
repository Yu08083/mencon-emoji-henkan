# 使い方ガイド

## 基本: Web版で暗号表を作る

1. [Web版](https://yu08083.github.io/mencon-emoji-henkan/) を開く
2. 右上の ⤴ ボタンをタップ → 「QRコード」または「URL」を取得
3. これを各プラットフォームに渡す

## Android で完全自動検知を有効化

1. アプリを起動 → トップに権限バナーが出る
2. **「通知へのアクセス」を許可** — LINEの通知を読むのに必要
3. **「他のアプリの上に表示」を許可** — フローティング窓のため
4. 完了。これでLINEに暗号メッセージが届くたびに自動でフローティング窓が出ます

### 暗号表をWeb版から取り込む

- 方法A: Web版のQRをアプリの右上 📷 ボタンで撮影
- 方法B: Web版で「URLをコピー」 → AndroidでURLを開く → アプリが自動起動して取り込み

## iOS で素早く復号

1. LINEで暗号メッセージを長押し
2. メニューから「共有」をタップ
3. 共有先一覧で **「メンコン絵文字で復号」** を選ぶ
4. 復号結果がモーダルで表示される

初回はShare Extensionが共有候補に出ないことがあります。「その他」から有効化してください。

## ブラウザ拡張で LINE Web を快適に

1. `extension/` を Chrome に読み込んで有効化
2. https://chat.line.biz/ または line.me を開く
3. 暗号メッセージは自動で翻訳バナーが下に付きます
4. メッセージ作成中、ひらがなで書いて `Alt + E` → その場で暗号に置換

## 暗号表の作り方のコツ

- 五十音すべてをパッと見で記憶しなくていい
- よく使う文字（あ、い、う、か、た、な等）だけ覚えれば十分復号できる
- 濁点は元の絵文字に `"`、半濁点は `'` が付くだけ
- 「特殊文字の変換」セクションで `!` `?` `🍣` などの追加も可能

## トラブルシューティング

### Android: 通知が来ても何も起きない

- LINEの通知設定で **「メッセージ内容を表示」** がONになっているか確認（OFFだと中身を読めない）
- 「通知へのアクセス」が**LINE通知の検知（メンコン絵文字）**になっているか再確認
- 一部端末では省電力モードでバックグラウンド処理が制限されます。バッテリー設定で「制限なし」に

### iOS: 共有メニューに出ない

- iOS設定 > 共有メニュー > 「その他」から「メンコン絵文字で復号」を有効化
- 一度Share Extensionをタップすれば、次回からは上位に表示されやすくなります

### ブラウザ拡張: バナーが付かない

- 拡張機能アイコンをクリックしてトグルがONになっているか確認
- LINE Web のDOMが更新されている可能性。GitHub Issues で報告お願いします
